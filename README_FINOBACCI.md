Fibonacci Calculator

Projeto desenvolvido para praticar conceitos de desenvolvimento web utilizando HTML, CSS e JavaScript.

Tecnologias utilizadas
HTML5
CSS3
JavaScript
Sobre o projeto

A aplicação realiza o cálculo da sequência de Fibonacci de forma dinâmica a partir de um número informado pelo usuário.

O projeto também foi utilizado para praticar:

Manipulação do DOM
Eventos de clique
Funções em JavaScript
Estruturação HTML
Estilização com CSS
Integração entre HTML, CSS e JS
Funcionalidades
Inserção de números pelo usuário
Cálculo automático da sequência de Fibonacci
Exibição dinâmica do resultado na tela
Interface simples e responsiva
